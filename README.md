## Vector Search Fundamentals

# Usage
1. Create an event trigger in MongoDB Atlas for inserting, updating, and replacing documents.
2. Import `event_trigger.js` as the trigger function.
3. Use `create_embeddings.py` to retrieve embeddings.
4. Use `vector_search_index.js` to create a vector search index.
5. Use `vector_search_query.js` to query the vector database.
